## 📖 Overview
**Classbit HRMS Portal** is a Human Resource Management System designed to help organizations manage employee information, attendance, grievances, and other HR processes.  
It is built with **Node.js** and **PostgreSQL**, featuring a modern and user-friendly interface.

---

## ✨ Features
- 👨‍💼 **Employee Management** – Profiles, dashboards, and records  
- ⏰ **Attendance Tracking** – Daily attendance and leave management  
- 📩 **Grievance Redressal** – Complaint submission and notification system  
- 📊 **Analytics Dashboard** – Visual representation of HR data  
- 🔒 **Secure Login** – Authentication and role-based access

## 🚀 Getting Started
### Prerequisites
- Node.js (v16+)
- MySQL
- Git

### Installation
```bash
# Clone the repository
git clone https://github.com/Sahil485661/Classbit-HRMS-Portal.git

# Navigate to project directory
cd Classbit-HRMS-Portal

# Install dependencies
npm install

# Start the server
npm start
# Start the Frontend
npm run dev
```
